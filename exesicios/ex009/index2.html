tipos de listas

lista oganizada

Em uma lista organizada podemos definir em três tipos de litas, numerica(1), alfabetica(a ou A),româna(i ou I) para adicionar uma lita basta abrir a tag ol, ela abrira um lista organizada, para por elemento na sua lista coloque a tag "li" em cada elemento de sua lista.Ex.:

  1. café
  2. pão
  3. leite
  4. manteiga

lista desoganizada

para uma lista desoganizada, ponha a tag "ul" para criar uma lista em "li" em cada elemento de sua lista.Essa lista também se divide em três tipos:circle,disc e square. uma lista sem "ul" sem colo o atributo "type", ela vira com disc por padrão E.:

lista anomal type disc

  • arroz
  • feijão
  • carne
  • macarrão

lista type circle

  • arroz
  • feijão
  • carne
  • macarrão

lista type square

  • arroz
  • feijão
  • carne
  • macarrão

juntando listas

Para junta as lista primiro abra a tag "ol" ou "ul" dependendo da sua lista, organizada ou não, apois crie uma elemento com "li", em baixo dele abra outra tag "ol" ou "ul", agora coloque outro elemento dentro dela, se elemento deve corresponde ao anterior.

Esse tipo de lista seve para dar um definição para um item que deseja.

  • Depois de baixor dela ponha a tag "dt" agora coloque o que quizer dar significado.
  • Agora abra a tag "dd" e coloque o significado do anterior.
